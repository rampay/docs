## How the API flow works

- Show list of supported fiat currencies and crypto currencies to your users using the available APIs
- Get pricing details for fiat amount, cryptocurrency and fiat currency.
- Create an order (using the API secret for auth) in your backend. That will return an order Id.
- Send the order Id along with your API key in the query params like this: [https://exchange-sandbox.rampay.io?orderId=&apiKey=](https://exchange-sandbox.rampay.io?orderId=&apiKey=)
- You can get the status of each order using the GET order API mentioned below.
