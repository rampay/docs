---
title: "Verify OTP"
description: "Verify OTP for verifying user's email address"
api: "POST /v1/partner/order/verify"
---

### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`

### Body

<Param body="orderId" type="string" required>
</Param>
<Param body="otp" type="string" required>
</Param>
<RequestExample>

```bash Request Example
curl --location --request POST 'https://api-sandbox.rampay.io/v1/partner/order/verify' \
--data-raw '{
    "orderId": "def0077c84211116d0a4c199850153c3",
    "otp": "333343"
}'
```

</RequestExample>
