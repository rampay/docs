---
title: "List of Crypto Currencies"
description: "Get a list of available crypto currencies"
api: "GET /v1/partner/crypto_currencies"
---

### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`
### Parameters

<Param query="countryCode" type="string" required>

Country code of the end user. This is required because we support different crypto currencies in different countries due to local regulations.

Example: `BR`

</Param>

<RequestExample>

```bash Request Example
curl --location \
  --request GET 'https://api-sandbox.payfura.com/v1/partner/crypto_currencies'
```

</RequestExample>
