---
title: "Create Order"
description: "Create a new order"
api: "POST /v1/partner/order"
---

### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`

### Body

<Param body="order" type="object" required>

<Expandable title="Toggle object">

<Param body="cfpmId" type="string" required>

This ID is part of List of fiat currencies API response. It helps us identify the fiat currency and the payment method that user wishes to pay with.

</Param>

<Param body="cryptoCurrencyId" type="string" required>

Crypto currency as part of List of crypto currencies response.

</Param>

<Param body="fiatAmount" type="string" required>

Amount of fiat currency

</Param>

<Param body="walletAddress" type="string" required>

Crypto wallet address for the selected crypto currency.

</Param>

<Param body="walletAddressMemo" type="string">

(Optional) For crypto currencies that support memo along with a wallet address, this field can be used to send the memo. Look for memo_required property in crypto currency list response

</Param>

<Param body="partnerMetadata" type="object">

(Optional) A key value map that can be used to place any client level data like client order id, email etc. This will be returned back in order get response.

</Param>

</Expandable>

</Param>

<Param body="userId" type="string">

(Optional) If sent, will take the user directly to payment screen

</Param>

```bash Request Example
        "fiatAmount": 3500,

        "partnerMetadata": {
          "orderId": "abc"
        }

```
