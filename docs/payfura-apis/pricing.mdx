---
title: "Pricing"
description: "Create price"
api: "POST /v1/partner/pricing"
---

### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`
### Body

<Param body="fiatAmount" type="integer" required>

Amount of fiat currency.

Example: `200`

</Param>

<Param body="cryptoCurrencyCode" type="string" required>

Crypto currency as part of List of crypto currencies response.

Example: `USDT`

</Param>

<Param body="cryptoCurrencyNetwork" type="string">

(Optional) The abbreviated code of the blockchain network. This needs to be included in crypto currencies like USDT that support different networks like erc20, trx etc.

Example: `erc20` (for ethereum)

</Param>

<Param body="direction" type="string">

Whether to buy or sell. Currently only BUY_CRYPTO is supported.

Example: `BUY_CRYPTO`

</Param>

<Param body="cfpmId" type="integer" required>

The fiat currency and the payment method that user wishes to pay with.

Example: `1`

</Param>

<RequestExample>

```bash Request Example
curl --location --request POST 'https://api-sandbox.payfura.com/v1/partner/pricing' \
  --data-raw '{
      "fiatAmount": 200,
      "cryptoCurrencyCode": "USDT",
      "cryptoCurrencyNetwork": "erc20",
      "direction": "BUY_CRYPTO",
      "cfpmId": 1
  }'
```

</RequestExample>
