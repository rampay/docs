---
title: "List of Fiat Currencies"
description: "Get a list of available fiat currencies"
api: "GET /v2/partner/fiat_currencies"
---
### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`

### Parameters

<Param query="countryCode" type="string">

(Optional) Country code of the end user. If sent you will get only the data for the given country.

Example: `US`

</Param>

<RequestExample>

```bash Request Example
curl --location \
  --request GET 'https://api-sandbox.payfura.com/v2/partner/fiat_currencies'
```

</RequestExample>
