---
title: "Get User"
description: "Fetch a user by their identifier"
api: "GET /v1/partner/user/{USER_ID}"
---

### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`
### Parameters

<Param path="USER_ID" type="string" required>

The User ID of the user to retrieve

Example: `f390fbe38f9646a05e293307b0f4097f`

</Param>

<RequestExample>

```bash Request Example
curl --location --request GET 'https://api-sandbox.payfura.com/v1/partner/user/f390fbe38f9646a05e293307b0f4097f'
```

</RequestExample>
