---
title: "Update User"
description: "Update a user profile with new data"
api: "PUT /v1/partner/user/{USER_ID}"
---
### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`

### Parameters

<Param path="USER_ID" type="string" required>

The user identifier of the user to retrieve

Example: `f390fbe38f9646a05e293307b0f4097f`

</Param>

### Body

<Param body="firstName" type="string">

First name of the user.

Example: `jack`

</Param>

<Param body="lastName" type="string">

Last name of the user.

Example: `doe`

</Param>

<Param body="email" type="string">

Email address of the user.

Example: `john@example.com`

</Param>

<Param body="dateOfBirth" type="string">

The birth day of the user in format `yyyy-mm-dd`

Example: `1990-12-22`

</Param>

<Param body="isdCode" type="string">

The [ISD Code](https://www.indyatour.com/directory/isd-code) (country calling code) of the user

Example: `+1`

</Param>

<Param body="phoneNumber" type="string">

Phone number of the user

Example: `5417543010`

</Param>

<Param body="addressInfo" type="object">

You can't update only a particular field inside address. If you are looking to update address, then complete address details need to be provided.

<Expandable title="Toggle object">

<Param body="addressLine1" type="string" required>

Address Line 1

Example: `Rue de Forte`

</Param>

<Param body="addressLine2" type="string">

Address Line 2

Example: `Louvre Road`

</Param>

<Param body="state" type="string" required>

The state of the user's address

> For United States, it is mandatory to send the `state` in 2 letter short code like CA, FL

Example: `CA`

</Param>

<Param body="city" type="string" required>

The city of the address

Example: `Los Angeles`

</Param>

<Param body="country" type="string" required>

The country (ISO 3166-1 alpha-2 code)

Example: `US`

</Param>

<Param body="postalCode" type="string" required>

The user's postal code

Example: `12345`

</Param>

</Expandable>

</Param>

<RequestExample>

```bash Request Example
curl --location --request PUT 'https://api-sandbox.payfura.com/v1/partner/user/f390fbe38f9646a05e293307b0f4097f' \
--data-raw '{
    "firstName": "Jack",
    "addressInfo": {
        "addressLine1": "street22",
        "addressLine2": "street2",
        "state": "Newyork",
        "city": "Gotham",
        "country": "DE",
        "postalCode": "21234"
    }
}'
```

</RequestExample>

</ResponseExample>
